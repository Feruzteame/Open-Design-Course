ABOUT THE PROJECT

This project is not aimed to change the previous website but to give an additional content about the 2019 Open design course participant by putting it into the already existed website . The content will mainly tell about what is Open Design Course using this website. The aims of this project are:
  • To explain who are the Open design course 2019 participants
  • To explain what we learned in the course in the hope that the upcoming participant can have a clear expectations towards the course
  • Make a platform as a memory for the open design school 2019 participants
This documentary project is a collaborative effort from all of the open design course 2019 participants.We would like to thank you the participants for the interview and engagement during the process.
